# Support

## Getting Help

For installation and usage questions, open a GitHub issue in this repository.

When reporting a problem, include:

- OS version (for this project: macOS expected)
- What command you ran
- Relevant log lines from `/tmp/claude-gateway.log` and `/tmp/claude-gateway.err`
- Steps to reproduce

## Security Issues

For security-sensitive reports, follow [SECURITY.md](SECURITY.md) and avoid public disclosure until triage.

## Conduct

Please follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) in all communications.
